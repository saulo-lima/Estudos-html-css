Minha evolução com HTML e CSS

Comecei a estudar programação no dia 02 de março de 2026. Desde então, venho aprendendo HTML e CSS através das aulas do Curso em Vídeo, com Gustavo Guanabara, além de utilizar o ChatGPT como apoio para tirar dúvidas e aprofundar meu entendimento de forma autodidata.

Durante minha jornada, já tive contato com diversos conceitos importantes, como semântica em HTML, uso de imagens, iframes, responsividade com media queries, além de propriedades de CSS como display, flexbox e grid.

Este projeto é uma iniciativa própria, criada com o objetivo de praticar, organizar meu aprendizado e aplicar, na prática, tudo o que venho estudando. Busco evoluir constantemente, desenvolvendo não apenas conhecimento técnico, mas também autonomia na resolução de problemas.

  • HTML Semântico (header, main, section, article, footer)
  • Imagens (img, picture, formatos e adaptação)
  • Links e navegação (a, navegação entre páginas)
  • iFrames (incorporar conteúdos externos)
  • Responsividade (mobile first)
  • Media Queries (adaptação para diferentes telas)
  • CSS básico (cores, fontes, medidas)
  • Box Model (margin, padding, border)
  • Display (block, inline, inline-block)
  • Flexbox (alinhamento e organização de layouts)
  • CSS Grid (estruturação de layouts mais complexos)
