Memória ram

A memória RAM (Random Access Memory ou Memória de Acesso Aleatório) é um componente volátil e de curtíssimo prazo que armazena temporariamente os dados dos programas e arquivos em uso, permitindo acesso rápido pelo processador (CPU). Ela garante agilidade ao computador, mas apaga todo o conteúdo ao ser desligada.
